生成式：一边循环一边计算的机制【适用于可能只使用某一部分的数据，不需要全部生成所有数据，节省内存空间】

打印生成式中的每一个元素，可以使用next()，没有更多的元素时，抛出`StopIteration`的错误。也可以使用for循环



普通函数是顺序执行，生成器是调用next()执行，遇到yield返回，再次执行从yield下一句开始执行

多次调用的generator函数是互相独立的generator
